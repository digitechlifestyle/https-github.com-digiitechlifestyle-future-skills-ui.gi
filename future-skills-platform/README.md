# Future Skills Schools Platform

A React/Vite project for an education platform teaching:

- AI Literacy
- Financial Literacy
- Digital Assets & Blockchain
- Business & Systems Thinking

## Upload to GitHub

1. Download and unzip this folder.
2. Open your GitHub repository in a browser.
3. Click **Add file** > **Upload files**.
4. Drag all files/folders from the unzipped project into GitHub.
5. Click **Commit changes**.

## Run locally

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```
