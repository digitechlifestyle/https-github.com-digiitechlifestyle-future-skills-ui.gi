# Future Skills Schools Platform

A professional React/Vite website prototype for a school-facing education platform teaching:

- AI Literacy
- Financial Literacy
- Digital Assets & Blockchain Safety
- Business & Systems Thinking

## What is included

- React app
- Vite setup
- Tailwind CSS setup
- SEO-friendly page content
- Teaching module pages
- Resource pages
- Pricing pages
- Teacher and student dashboard previews

## How to run locally

```bash
npm install
npm run dev
```

## How to upload to GitHub

1. Unzip this folder.
2. Go to your GitHub repository.
3. Click **Add file**.
4. Click **Upload files**.
5. Drag all unzipped files into GitHub.
6. Click **Commit changes**.

## Important

Do not upload `node_modules`. This project already includes a `.gitignore` file.
